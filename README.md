# creeper-jump
This is an minecraft/underground themed platformer that uses godot. There is several traps in the level, and there is a giant starcase up. To play, just go to the itch.io link (https://kanokid.itch.io/creeperjumper), it auto loads and you can use Space for jump, A and Left arrow for moving left, and D and Right arrow for moving right.

## AI Usage:
I had the Godot AI MCP extension installed because I was planning to use it, I never got it setup though, but I did use some AI for some help on seeing how to change gravity and run speed.
